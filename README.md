# 韩菁芸｜AI 产品作品集

这是我的产品作品集，用 5 个项目展示我在 AI 产品经理方向上的能力：需求拆解、Agent / Skill 产品化、规则建模、轻量增长设计和原型表达。

## 项目总览

1. `Campus Apply Skill`
   - 校招投递辅助工作流，强调自动化边界与风险控制。

2. `AI-KP 飞书群跑团主持机器人`
   - 飞书群里的垂直 Agent，强调消息入口、知识检索、权限分层和场景推进。

3. `Vibe Fitness Pro`
   - 个人健身复盘工具，强调训练记录、身体变化和知识沉淀的一体化。

4. `双身份狼人杀发牌工具`
   - 桌游规则辅助工具，强调规则建模和约束校验。

5. `酒瘾人格测试`
   - 社交传播型轻娱乐测试，强调分享属性和稳定评分逻辑。

## 能力总结

- 能把真实场景拆成明确的产品问题，并找到最小可行闭环。
- 能把 AI、规则、工作流和权限控制结合成可落地的产品方案。
- 能从用户效率、风险控制和传播体验三个角度同时做设计取舍。
- 能把产品逻辑、技术实现和视觉表达放进同一套作品集叙事里。

## 作品集表达重点

- 前 5 个项目分别对应不同类型的产品问题与解决路径。
- 每个项目都强调了“为什么做”“怎么做”“为什么这样做”。
- 整体目标是展示我作为 AI 产品经理的综合判断力，而不只是功能堆叠。
