# 韩菁芸｜AI 产品作品集

这是我的产品作品集素材包，目标是用 5 个项目展示我在 AI 产品经理方向上的能力：需求拆解、Agent / Skill 产品化、规则建模、轻量增长设计和原型表达。

## 作品集包含什么

- `index.html`：对外展示版 HTML 作品集
- `index.pdf`：便于分享和投递的 PDF 版本
- `deck_content.md`：逐页内容脚本
- `diagrams/`：流程图、思维导图、架构图的 Mermaid 源文件
- `assets/screenshots/`：项目截图与封面图

## 五个项目怎么体现我的能力

1. `Campus Apply Skill`
   - 体现我对 AI Skill 产品化和安全边界的理解。
   - 重点是把自动化控制在用户可确认的范围内。

2. `AI-KP 飞书群跑团主持机器人`
   - 体现我对垂直 Agent 场景的产品拆解能力。
   - 重点是消息入口、资料检索、权限分层和状态推进。

3. `Vibe Fitness Pro`
   - 体现我对复杂个人行为场景的整合能力。
   - 重点是训练记录、身体数据、照片对比和知识沉淀的闭环。

4. `双身份狼人杀发牌工具`
   - 体现我把现实规则抽象成约束模型的能力。
   - 重点是把“发牌”这种重复操作变成可执行的产品逻辑。

5. `酒瘾人格测试`
   - 体现我对传播型产品和轻增长体验的理解。
   - 重点是短路径、可分享结果和稳定评分逻辑。

## 我的方法论

- 先找真实场景里的高频痛点，再决定要不要做 AI。
- 把复杂问题拆成可验证的最小闭环。
- 在自动化、可控性和用户体验之间找平衡。
- 让结果页、流程图和技术方案都能服务产品表达。
