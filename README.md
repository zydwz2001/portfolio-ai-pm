# 韩菁芸｜产品作品集素材包

这个文件夹用于整理产品作品集 HTML PPT 和相关视觉素材。

## 文件结构

- `index.html`：对外展示版 HTML PPT。
- `deck_content.md`：逐页内容脚本。
- `diagrams/`：流程图、思维导图、架构图的 Mermaid 源文件。
- `assets/screenshots/`：项目截图与封面截图。
- `asset_checklist.md`：后续建议补充的视觉素材。

## 当前作品集结构

1. Campus Apply Skill：校园招聘投递辅助 Codex Skill。
2. AI-KP 飞书群跑团主持机器人：群聊场景下的垂直 AI Agent。
3. My Keep：个人健身记录与复盘工具。
4. 双身份狼人杀发牌工具：线下聚会玩法的规则辅助工具。
5. 酒瘾人格测试：朋友小圈层传播的轻娱乐测试网站。
6. 项目能力总结：AI 产品经理匹配度。

## 打开方式

```bash
open index.html
```

或：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。
